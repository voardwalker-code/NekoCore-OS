🔤 IDs
Chunks stored with doc_<hash> prefix. Excluded from subconscious context to prevent noise flooding (E1). Excluded from chatlog recall ltmFolders scan (E2).
🗜️ Compression
Each chunk is V4-compressed (zlib) on disk. Reconstruct via getChatlogContent(). Memory schema v1 normalization is applied on write.
📡 Recall
Stored for reconstruction and long-term knowledge. `doc_*` chunks are intentionally excluded from subconscious live-turn context to prevent retrieval noise. Preview endpoint: /api/memory/preview

🏗️ Architecture Principles
Contract-First
Memory schema v1 · contributor contracts · response contracts enforced on every turn.
Event-Driven
CognitiveBus broadcasts all thoughts. SSE streams them to the visualizer in real-time.
Phase Isolation
Brain loop uses per-phase try/catch + health counters + circuit-breaker. No phase can crash the loop.
File-First Persistence
Everything is plain JSON on disk. No database dependency. Portable, inspectable, diff-able.
Multi-Model LLM Routing
ModelRouter selects different models for different phases. Budget guard prevents runaway token usage.
Pluggable Skills
Skills are drop-in folders. Trigger-based activation via /skill and on-demand /tool docs loading.
