📖 How Skills Work

A skill teaches the entity what it can do and how to do it. Skills are only loaded into the prompt when explicitly invoked — they are never injected by default.

Invoking a skill in chat: type /skill <trigger> your request here. The trigger is the exact text set in the Trigger field below — it is case-sensitive and exact. Only a perfect match activates the skill. For example: /skill web-search find me articles on memory consolidation

Workspace tools (ws_read, ws_write, etc.) are loaded separately with /tool anywhere in your message.

Tools inside a skill are defined as ### tool_name headings under a ## Tools section. The entity calls them using [TOOL:tool_name param="value"] syntax.

Browse community skills at ClawHub.ai — download SKILL.md files and install them via the quarantine system.

If left blank, the skill name is used as the trigger. Avoid spaces — use hyphens. The match is exact and case-sensitive.
