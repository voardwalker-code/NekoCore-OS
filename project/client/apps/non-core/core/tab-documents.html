How It Works
  • Documents are automatically split into ~1500 token chunks
  • Each chunk is stored as both a conscious LTM memory and an episodic memory
  • Topics are extracted from capitalized words and filename
  • Sequential chunks are connected via the trace graph
  • Knowledge becomes fully searchable and retrievable during conversations
