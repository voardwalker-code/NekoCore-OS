Configure an LLM Provider

Before you can create or load entities, you need to set up an API connection. Choose OpenRouter (cloud-based), Anthropic Direct (Claude + prompt caching), or Ollama (local AI).

OpenRouter: Cloud-based, supports GPT-4, Claude, Llama, and more. Requires API key.
Anthropic Direct: Claude models with prompt caching — up to 90% savings on repeated system prompts.
Ollama: Local AI models, free and private. Requires Ollama running on your machine.
